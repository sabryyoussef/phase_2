Employee Salary Certificate
This module helps you to create a record for salary certificate and you can print it as PDF.

Key Features:

  • Create as a record It allows you to create a salary record.
  • Generate a PDF When an employee's salary information is created and by clicking salary certificate print menu it will generate a PDF.
  • Non Delete Once a salary certificate is created no one can delete it.
  • Dynamic data The data which is filled in the employees salary record the name, DOJ, Purpose of the certificate, Date of Issue etc will be update dynamically to the PDF.

How it works:

Employee -> Certificates

After module installation a new menu will be displayed:


Click the Salary Certificates Menu
Salary Certificate records list view

Create Employee's Salary certificate details as a record

Go to the action then Click the Salary certificate print
Certificate Print
