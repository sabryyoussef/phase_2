Freezoner Custom Views Documentation

Overview

This documentation covers the customizations made to various Odoo views in the Freezoner module. These customizations enhance the standard Odoo functionality with additional features and improved workflows.

1. Task Management (task.xml)

Enhanced Kanban View

Added a "View Project" quick access link in the task kanban view for better navigation.

<a t-if="record.project_id.raw_value"
   t-attf-href="/web#id={{record.project_id.raw_value}}&model=project.project&view_type=form"
   class="o_kanban_record_title"
   style="font-weight: bold; color: green;">
    View Project
</a>

Document Management Integration

  • Added document upload and management directly in tasks
  • Document expiration tracking
  • Document verification workflow
  • Partner-specific document requirements

Use Case: Document Collection Workflow

  1. Create a task for document collection
  2. Specify required documents with expiration dates
  3. Upload and verify documents
  4. Track document status and completion
  5. Move task to next stage when all documents are verified

Technical Implementation

The customizations are implemented through XML view inheritance, extending the standard Odoo views with additional functionality. Key technical features include:

  • Custom status bar widgets for stage management
  • Document management integration with binary file handling
  • Partner-specific document requirements tracking
  • Enhanced kanban views with quick access features
  • Custom action buttons and workflows

Benefits

  • Streamlined document management workflow
  • Improved project-task navigation
  • Enhanced stage management
  • Better document verification process
  • Improved user experience with quick access features
  • Comprehensive partner and document tracking
