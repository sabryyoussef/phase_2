HR Attendance Geofence

The HR Attendance Geofence module allows HR managers to create virtual geographic boundaries (geofences) for attendance locations. Employees can only check in and out within these geofenced areas.

Key Features

  • Create geofences for attendance locations.
  • Restrict check-ins and check-outs to specific geographic areas.
  • Integrates seamlessly with Odoo's HR Attendance module.
