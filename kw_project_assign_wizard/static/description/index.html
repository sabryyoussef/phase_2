Description of the Module

This module will help you to mass assign a Contractor and mass change the task stage separately by checking or unchecking the corresponding box. Or you can unassign the Contractor if you leave the field blank.

After installing this module, go to the Project / My tasks menu. After selecting the desired tasks, click Action and select Assign wizard.

Next, we can specify the stage of the task and the Contractor (or Contractors). Click the Confirm button.
